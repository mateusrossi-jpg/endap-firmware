Nota prática

Como usar

GPIO nativoContinua sujeito aos bloqueios de Wi‑Fi, RS485, W5500 e conflitos com outros bindings.
MCP23x17Usa instância e canal A0..B7. O binding já aceita esse backend; a aplicação elétrica/runtime é a próxima camada.
RS485 / EthernetContinuam reservando pinos do transporte local. O preview automático afeta apenas o backend GPIO.
Input learningSegue valendo para entradas físicas locais. Canais MCP entram pela configuração de backend/endereço.
